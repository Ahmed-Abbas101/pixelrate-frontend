## PixelRate - Frontend
The client-side interface for PixelRate, a full-stack web application designed to help users browse, compare, and evaluate mobile specifications, prices, and benchmark scores. Built with a focus on responsive design and a seamless user experience.

- React
- Tailwind
- CSSNode

## Key Features
- Dynamic Spec Sheets: Cleanly rendered mobile specifications (Display, Processor, Battery, etc.).
- Advanced Filtering & Sorting: Easily find devices by brand, price range, or processor score.
- Responsive Design: Fully functional and visually consistent across desktop, tablet, and mobile screens.
- Real-time Data Fetching: Integrates seamlessly with the PixelRate REST API.

## Tech Stack
- Core Library: React.js (Vite / Create React App)
- Styling: Tailwind CSS / CSS Modules
- Routing: React Router DOM
- HTTP Client: Native Fetch API
- Icons: React Icons
